# qcircuit-relayer
a simple and reusable python boilerplate module for platform-agnostic job execution/simulation from ""any"" Qiskit-native application.


# Acknowledgements
This project was inspired by previous work - [OSC-Qasm](https://github.com/Quantumland-art/OSC-Qasm) (clear separation betwen circuit design vs simulation and execution) and [VQH](https://github.com/iccmr-quantum/VQH) (the 'hardware/' folder is essentially a first iteration of this project, which is now being refactored here).
